# Grocery Store Web Application

A simple grocery shopping website built using PHP and MySQL.

## Features
- User Registration and Login
- Browse Products by Category
- Add to Cart and Wishlist
- Checkout system
- Admin dashboard to manage products and users

## Technologies Used
- PHP
- MySQL (phpMyAdmin)
- HTML
- CSS
- JavaScript
- XAMPP

## How to Run the Project

1. Install XAMPP
2. Copy the project folder to:

C:\xampp\htdocs\

3. Start Apache and MySQL in XAMPP

4. Import database file `shop_db.sql` in phpMyAdmin

5. Open the project in browser:

http://localhost/grocery_store/

## Admin Panel

http://localhost/grocery_store/admin_page.php
